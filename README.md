# StatisticalSystemOfAnnualReportPlatform
河北省创新年报平台统计系统

技术栈：Jsp+Servlet+Oracle 
项目描述：近年来，各行业对年报的填报工作量增多，需要简易的填写操作。 随着企业机构的增加，对数据的查找和统计更加复杂，需求量大，审核工作也更加难以进行，本系统针对这些问题进行了设计
